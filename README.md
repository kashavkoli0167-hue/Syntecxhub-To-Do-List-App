# Syntecxhub-To-Do-List-App(FocusList)

A simple **To-Do List web app** built with **HTML, CSS, and JavaScript** as part of my **Frontend Internship at SyntaxHub**.  
It helps users manage daily tasks with an interactive UI and persistent storage.

## Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks individually  
- Mark tasks as completed ✅  
- Filter tasks: All / Completed / Pending  
- Dark / Light mode toggle 🌙☀️  
- Persistent storage using **LocalStorage**  
- Responsive design for desktop & mobile  

## Tech Stack

- **HTML** – Structure of the app  
- **CSS** – Styling, responsive design, dark/light mode  
- **JavaScript** – DOM manipulation, event handling, LocalStorage
